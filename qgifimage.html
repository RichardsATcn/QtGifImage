void QGifImage::addFrame(const QImage & frame, int delay = -1)

Append the QImage object frame with delay.

As gif file only support indexed image, so all the frame will be converted to the QImage::Format_Indexed8 format. Global color table will be used in the convertion if it has been set.

QImage::offset() will be used when insert the QImage to the gif canvas.

void QGifImage::setDefaultTransparentColor(const QColor & color)

Set the default transparent color.

Unlike other image formats that support alpha (e.g. png), GIF does not support semi-transparent pixels. The way to achieve transparency is to set a color that will be transparent when rendering the GIF. So, if you set the transparent color to black, the black pixels in the gif file will be transparent.

See also defaultTransparentColor().

void QGifImage::setFrameTransparentColor(int index, const QColor & color)

Sets the transparent color of the frame index. Unlike other image formats that support alpha (e.g. PNG), GIF does not support semi-transparent pixels. The way to achieve transparency is to set a color that will be transparent when rendering the GIF. So, if you set the transparent color to black, the black pixels in your gif file will be transparent.

See also frameTransparentColor().

void QGifImage::setGlobalColorTable(const QVector<QRgb> & colors, const QColor & bgColor = QColor())

Set the global color table colors and background color bgColor. bgColor must be one the color in colors.

Unlike other image formats that support alpha (e.g. png), GIF does not support semi-transparent pixels. So the alpha channel of the color table will be ignored.

See also globalColorTable().

void QGifImage::setLoopCount(int loop)

Set the loop count. The default value of loop is 0, which means loop forever.
