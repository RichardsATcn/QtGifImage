Qt GifImage

The Qt GifImage Module provides a set of classes to read and write .gif files. The library can be used to

  • Generate a new .gif file from a list of QImage files.
  • Extract frames to a list of QImage files.

Source code:https://github.com/dbzhang800/QtGifImage
Issures:https://github.com/dbzhang800/QtGifImage/issues
License:MIT

Getting Started

To include the definitions of the module's classes, using the following directive:

#include <QtGifImage>

To link against the module, add this line to your qmake .pro file:

QT += gifimage

More information can be found in Qt GifImage Build page.
